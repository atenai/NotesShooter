# NotesShooter

「ノーツシューター」

（概要・テーマ）
この作品は、デジタルハリウッド大学院の、ゲームラボという授業で開発した作品です。
元々は、HAL時代にWii向けに開発した的当てゲームを、
ビートセイバーのような音ゲーとしてリメイクするというテーマで制作しました。

(開発時期・期間)
デジタルハリウッド大学院1年生時の2020年6月～2020年12月(7ヶ月)

（開発ツール）
・ゲームエンジン：Unity(バージョン2019.1.4f1)
・言語：C# 
・開発環境：Visual Studio2017 
・3D データ制作： Blender
・2D データの一部はフリー素材を使用しています。
・BGMは著作権フリーの「NoCopyRightSounds」を使用しています。
・SE はフリー素材を使用しています。 

（チームメンバー）
開発チーム（計3人）
・プログラマー1人（担当）
・シェーダー１人
・デザイナー１人

（担当内容）
自分はディレクターとプログラマーとプランナーの3つを兼任しました。
プログラムはシェーダー以外の全てを制作しました。
プランニングは企画書の作成・レベルデザインをしました。
マネジメントはチームの進捗状況の管理をやりました。

（ゲームシステム）
出現するノーツを撃っていく音ゲーシューターです。
・ゴールにたどり着くとリザルト画面にシーン遷移します。
・キューブを壊すごとにスコアが上がります。
・赤キューブは赤弾で壊す事ができます。
・青キューブは青弾で壊す事ができます。
・紫キューブは赤弾・青弾のどちらかを5回当てると壊す事ができます。
・ドラム缶に弾を当てると周辺のキューブをまとめて壊す事ができます。
・黒い壁は、弾で壊す事ができます。

(操作方法)
[キーボード]
マウス：カメラ
左クリック：青弾
右クリック：赤弾
Q：青弾リロード
E：赤弾リロード
ESC：ゲーム終了

(起動方法)
「NotesShooter(ビルド)」というフォルダーを開いてください。
その中にある、「gamerabo FPS Project.exe」というアプリケーションをダブルクリックしてください。
